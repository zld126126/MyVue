## CLI 工具安装
    npm install -g @tarojs/cli

## 查看 Taro 全部版本信息
    npm info @tarojs/cli

## 项目初始化
    taro init myApp
    安装依赖:
        cd myApp && cnpm install

## 项目运行:
    npm run dev:weapp
    npm run dev:h5
